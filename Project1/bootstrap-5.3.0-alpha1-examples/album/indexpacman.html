PacMan Project

This project was one of the first animations that I worked with. It is basically four images of the PacMan: open and closed mouth facing right and the same facing left. The program is set to check the bounds of the page so the PacMan can move from one side to the other no matter the size of the page. The program uses logic commands and a timer.
