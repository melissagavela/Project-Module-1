Map Animation Project

In this project I had started to use maps with mapbox. It was my first time using access tokens which was very interesting to me. It was simpler thant it may seem. The logic used a simple timer and counter to move through a list of coordinates. The marker moves through the list making the map animated. It shows the stops the bus makes between the Harvard and MIT campuses when you click on the "Show stops" button.
